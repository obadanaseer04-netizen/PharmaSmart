# 💊 PharmaSmart

PharmaSmart is a smart pharmacy management system designed to simplify pharmacy operations and improve efficiency.

---

## 🚀 Features

- Medicine inventory management
- Prescription handling
- Sales tracking
- Reports and analytics
- User-friendly interface

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL

---

## 📌 Project Goal

The goal of PharmaSmart is to digitalize pharmacy workflow and make medicine management easier and faster.
